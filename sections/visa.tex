\chapter{ویزای تحصیلی}
\label{ch:visa}

\section{مراحل کلی}

پس از دریافت \lr{Letter of Acceptance} یا معادل آن:

\begin{enumerate}
  \item جمع‌آوری مدارک مالی (\lr{Proof of Funds})
  \item پرداخت هزینه درخواست ویزا
  \item تکمیل فرم آنلاین
  \item آپلود مدارک و رزرو وقت بیومتریک/مصاحبه
  \item پیگیری تا صدور ویزا
\end{enumerate}

\section{مدارک رایج}

\begin{checklistbox}
  \begin{checklist}
    \item پاسپورت معتبر (حداقل ۶ ماه اعتبار)
    \item نامه پذیرش دانشگاه (\lr{LOA / CAS / I-20})
    \item گواهی تمکن مالی (گردش حساب، فیش بلوکه، ...)
    \item مدرک زبان
    \item عکس پرسنلی استاندارد
    \item فرم‌های پزشکی (در بعضی کشورها)
    \item \lr{GIC} (برای ویزای تحصیلی کانادا)
  \end{checklist}
\end{checklistbox}

\section{تمکن مالی}

\begin{infobox}[title={نکات مهم}]
  مبلغ مورد نیاز بسته به کشور، شهر و مدت تحصیل متفاوت است.
  منابع رسمی را ببینید — مثلاً برای کانادا:
  \weblink{حداقل مبلغ \lr{GIC}}{https://www.canada.ca/en/immigration-refugees-citizenship/services/study-canada/study-permit/get-documents.html}
\end{infobox}

\begin{warningbox}[title={هشدار درباره مدارک مالی}]
  از خرید مدارک جعلی خودداری کنید. سفارت‌ها گردش حساب و
  منبع پول را بررسی می‌کنند و پیامد رد ویزا یا ممنوعیت
  ورود بسیار سنگین است.
\end{warningbox}

\section{مصاحبه ویزا}

\begin{tipbox}
  در مصاحبه کوتاه و صادق باشید. آماده پاسخ به این سؤالات باشید:
  \begin{itemize}
    \item چرا این کشور و این برنامه؟
    \item بعد از تحصیل چه برنامه‌ای دارید؟
    \item چه کسی هزینه تحصیل را تأمین می‌کند؟
  \end{itemize}
\end{tipbox}
